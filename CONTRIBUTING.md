# Contributing to Awesome DAS Collection

First off, thank you for considering contributing! It's people like you that make this such a great resource for the community.

## How Can I Contribute?

### Suggesting a Resource
If you have a resource (paper, dataset, tool, etc.) that you think would be a great addition, please follow these steps:

1.  **Search existing issues and pull requests** to see if the resource has already been suggested.
2.  If it hasn't, please **open a new issue** using the "Resource Suggestion" template.
3.  Fill out the template with as much information as possible.

### Making a Direct Contribution via Pull Request
If you'd like to add a resource directly, please follow these steps:

1.  **Fork the repository** to your own GitHub account.
2.  **Create a new branch** for your changes (e.g., `add-new-dataset`).
3.  **Add your resource** to the `README.md` file, following the existing format:
    - Use the format: `[Resource Name](link) - A brief description.`
    - Add the entry to the most relevant category.
    - Keep the list sorted alphabetically within each category.
4.  **Check your spelling and grammar.**
5.  **Commit your changes** with a clear and descriptive commit message.
6.  **Push your branch** to your fork.
7.  **Open a pull request** to the main repository. Please use the pull request template.

## Contribution Guidelines

-   Ensure the resource is relevant to Distributed Acoustic Sensing (DAS).
-   Provide a working link.
-   Avoid broken links or paywalled content where possible (unless it's a significant and well-known resource).
-   Keep descriptions concise and to the point.

Thank you for your contribution!
